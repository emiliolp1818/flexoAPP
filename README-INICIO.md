# 🚀 FlexoApp - Sistema de Gestión de Empaques Flexibles

## 🎯 INICIO RÁPIDO (1 CLIC)

### Para usuarios nuevos:
```
1. Doble clic en: INICIAR-FLEXOAPP.bat
2. Esperar a que se abra el navegador
3. ¡Listo! La aplicación está funcionando
```

## 📋 ARCHIVOS IMPORTANTES

| Archivo | Descripción |
|---------|-------------|
| `INICIAR-FLEXOAPP.bat` | ⭐ **PRINCIPAL** - Inicia todo automáticamente |
| `verificar-sistema.bat` | 🔍 Verifica que todo esté instalado correctamente |
| `limpiar-reinstalar.bat` | 🧹 Limpia y reinstala si hay problemas |
| `diagnostico.bat` | 🔬 Genera reporte detallado para soporte |

## 🛠️ SOLUCIÓN DE PROBLEMAS

### ❌ Si no inicia:
1. Ejecutar: `verificar-sistema.bat`
2. Si hay errores, seguir las instrucciones mostradas
3. Si persiste, ejecutar: `limpiar-reinstalar.bat`

### ❌ Si hay errores extraños:
1. Ejecutar: `diagnostico.bat`
2. Revisar el archivo de log generado
3. Enviar el log si necesitas soporte

### ❌ Si los puertos están ocupados:
- Cerrar otras aplicaciones que usen puertos 4200 o 5000
- Reiniciar el sistema si es necesario

## 🌐 URLS DE LA APLICACIÓN

- **Frontend (Angular)**: http://localhost:4200
- **Backend (API)**: http://localhost:5000
- **Prueba de API**: http://localhost:5000/api/test

## 📊 FUNCIONALIDADES

### ✅ Dashboard
- Estadísticas en tiempo real
- Resumen de máquinas activas
- Navegación a todas las secciones

### ✅ Gestión de Máquinas
- 11 máquinas (números 11-21)
- Estados: activa, mantenimiento, parada
- Eficiencia y horas de operación

### ✅ Programas de Producción
- Crear, editar, duplicar, eliminar
- Estados: listo, corriendo, suspendido, terminado
- Paleta visual de colores
- Información completa del producto

### ✅ Sincronización
- Cambios guardados automáticamente
- Actualizaciones en tiempo real
- Funciona en múltiples pestañas/dispositivos

## 🔧 REQUISITOS DEL SISTEMA

- **Windows 7/8/10/11**
- **Node.js 16+** (se verifica automáticamente)
- **4 GB RAM mínimo**
- **Puertos 4200 y 5000 disponibles**

## 📞 SOPORTE

### Problemas comunes:

1. **"Node.js no encontrado"**
   - Descargar desde: https://nodejs.org/
   - Reiniciar el sistema después de instalar

2. **"Puerto en uso"**
   - Cerrar otras aplicaciones
   - Reiniciar el sistema

3. **"Error de dependencias"**
   - Ejecutar: `limpiar-reinstalar.bat`

4. **"No se conecta al backend"**
   - Verificar que no haya firewall bloqueando
   - Ejecutar como administrador si es necesario

### Para soporte técnico:
1. Ejecutar `diagnostico.bat`
2. Enviar el archivo de log generado
3. Describir el problema específico
